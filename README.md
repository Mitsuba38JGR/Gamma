# Cloud AI Memo with Auth

ユーザー登録・ログイン機能を備えたAIメモアプリです。
データはGoogleスプレッドシートに保存されるため、どのデバイスからでも自分の履歴にアクセスできます。

## 特徴
- **ユーザー認証**: ID（英数ひらがな）とパスワード（英数）による管理。
- **データ分離**: ログインしたユーザーのデータのみを表示します。
- **永続化**: スプレッドシート保存のため、アプリを再起動してもデータは消えません。

## セットアップ

1. **Google Cloud Platform (GCP)**
   - API有効化 (Sheets API, Drive API)
   - サービスアカウントキー(JSON)の取得

2. **スプレッドシートの準備**
   - 新規スプレッドシートを作成し、名前を `ai_memo_auth_db` に変更。
   - サービスアカウントのメールアドレスを「編集者」として招待。
   - **(自動作成されますが)** シート作成の手間を省くため、プログラムが初回起動時に `users` と `logs` シートを自動生成する仕組みになっています。

3. **環境変数の設定 (Streamlit Cloud)**
   `Secrets` に以下を設定してください。

   ```toml
   [openai]
   api_key = "sk-..."

   [gcp_service_account]
   type = "service_account"
   ... (JSONの中身)
